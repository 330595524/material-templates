# 物料模板

## 使用模板创建物料

```bash
$ tnpm i -g iceworks

$ mkdir my-materials && cd my-materials
# init material collection project
$ iceworks init material <templateNpmName>
# init sindle component project
$ iceworks init component <templateNpmName>
```
